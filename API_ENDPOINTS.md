API Endpoints

1. Upload Document
POST /documents/upload

Description:
Uploads documents to be processed by the RAG system.

2. Ask Question
POST /chat

Description:
Returns answers using retrieved document context.

3. GitHub Activity Summary
GET /integration/github-summary

Description:
Fetches repository activity and summarizes it.

4. Analyze Incident Logs
POST /agent/analyze

Description:
Analyzes operational logs and returns security insights.
